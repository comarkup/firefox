CoMarkup Framework Detection Test

This page contains code examples for different frameworks to test the CoMarkup extension's detection and preview functionality.

React Component Example

A simple React component demonstrating state management and event handling.

function Counter() {
    const [count, setCount] = React.useState(0);
    
    return (
        <div style={{ textAlign: 'center', padding: '20px' }}>
            <h2>Count: {count}</h2>
            <button 
                onClick={() => setCount(count + 1)}
                style={{
                    padding: '10px 20px',
                    fontSize: '16px',
                    background: '#007bff',
                    color: 'white',
                    border: 'none',
                    borderRadius: '4px',
                    cursor: 'pointer'
                }}
            >
                Increment
            </button>
        </div>
    );
}

Vue.js Component Example

A Vue.js component showing reactive data and methods.

const app = Vue.createApp({
    setup() {
        const message = Vue.ref('Hello Vue!');
        const count = Vue.ref(0);
        
        const increment = () => {
            count.value++;
        };
        
        return () => Vue.h('div', { style: { textAlign: 'center', padding: '20px' } }, [
            Vue.h('h2', message.value),
            Vue.h('p', `Count: ${count.value}`),
            Vue.h('button', {
                onClick: increment,
                style: {
                    padding: '10px 20px',
                    fontSize: '16px',
                    background: '#42b883',
                    color: 'white',
                    border: 'none',
                    borderRadius: '4px',
                    cursor: 'pointer'
                }
            }, 'Increment')
        ]);
    }
});

Vanilla JavaScript Example

A pure JavaScript example showing DOM manipulation and event handling.

const container = document.createElement('div');
container.style.textAlign = 'center';
container.style.padding = '20px';

const heading = document.createElement('h2');
heading.textContent = 'Vanilla JS Counter';
container.appendChild(heading);

const countDisplay = document.createElement('p');
let count = 0;
countDisplay.textContent = `Count: ${count}`;
container.appendChild(countDisplay);

const button = document.createElement('button');
button.textContent = 'Increment';
button.style.cssText = `
    padding: 10px 20px;
    font-size: 16px;
    background: #f7df1e;
    color: black;
    border: none;
    border-radius: 4px;
    cursor: pointer;
`;

button.addEventListener('click', () => {
    count++;
    countDisplay.textContent = `Count: ${count}`;
});

container.appendChild(button);

Svelte Component Example

A Svelte component demonstrating reactive declarations and event handling.

<script>
    let count = 0;
    
    function handleClick() {
        count += 1;
    }
</script>

<div style="text-align: center; padding: 20px;">
    <h2>Count: {count}</h2>
    
    <button on:click={handleClick}>
        Increment
    </button>
</div>

<style>
    button {
        padding: 10px 20px;
        font-size: 16px;
        background: #ff3e00;
        color: white;
        border: none;
        border-radius: 4px;
        cursor: pointer;
    }
    
    button:hover {
        background: #ff2d00;
    }
</style>

Angular Component Example

An Angular component showing data binding and event handling.

@Component({
    selector: 'app-counter',
    template: `
        <div style="text-align: center; padding: 20px;">
            <h2>Count: {{ count }}</h2>
            <button (click)="increment()">
                Increment
            </button>
        </div>
    `,
    styles: [`
        button {
            padding: 10px 20px;
            font-size: 16px;
            background: #dd1b16;
            color: white;
            border: none;
            border-radius: 4px;
            cursor: pointer;
        }
    `]
})
export class CounterComponent {
    count = 0;
    
    increment() {
        this.count++;
    }
}
